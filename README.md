# AI Digital Wellness System

## Project Overview

AI Digital Wellness System is an Android-based application that helps users maintain healthy digital habits. The system monitors social media usage, tracks screen time, sends notifications when usage exceeds limits, and uses Artificial Intelligence to analyze user behavior patterns.

## Objectives

* Monitor social media usage
* Track daily and weekly screen time
* Send smart notifications and alerts
* Detect excessive usage patterns
* Predict potential addiction risk using AI

## Features

### User Features

* Screen-time tracking
* App usage monitoring
* Daily and weekly reports
* Smart notifications
* Late-night usage alerts

### AI Features

* Usage pattern analysis
* Addiction risk prediction
* Personalized recommendations
