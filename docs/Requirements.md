# Requirements

## Functional Requirements

1. User Registration and Login
2. Screen Time Monitoring
3. Social Media Usage Tracking
4. Notification Generation
5. Daily and Weekly Reports
6. AI-Based Usage Analysis
7. Addiction Risk Prediction

## Non-Functional Requirements

1. User-Friendly Interface
2. Fast Response Time
3. Secure Data Storage
4. Reliable Notifications
5. Scalability

## Software Requirements

* Android Studio
* Java/Kotlin
* Firebase
* Python
* GitHub

## Hardware Requirements

* Android Smartphone
* Internet Connection
* Computer for Development
